White Space

Samsung

samsung.com

Samsung use white space and clean design quite well in this mobile screenshot. Samsung place images and headings well enough to draw the users attention to where they would like it to be. Since it's so clean with minimum distractions, the user is guided quite well to where Samsung would like their users to go.

Fitt's Law

The Courier Guy

thecourierguy.co.za

The Courier Guy uses Fitt's Law in this mobile screenshot because of the small distance between the selectable options, thus requiring less time for the user to select interactive objects. Some selectable objects are bigger making it easier to select.

Visual Hierarchy

Debonairs Pizza

debonairspizza.co.za

Debonairs Pizza use visual hierarchy quite well in this mobile screenshot. The first thing that you notice is the large pizza image. The next thing you see is the second largest item which is the word "AMAZING". You then see the rest of the webpage which is designed to make the viewer want their product through visual hierarchy.
